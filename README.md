# Secure Sidekick

AI-powered scam and phishing message checker built with Next.js, React, Supabase, OpenAI, Stripe, and Vercel.

Live site: https://www.securesidekick.org

## Features

- AI-powered suspicious message analysis
- Plain-English scam explanations
- Magic-link authentication
- Stripe subscription billing
- Usage tracking
- Responsive UI
- Trust-focused product design

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI API
- Stripe
- Vercel
